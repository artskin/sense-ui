🖖SenseUI is a Web Components &amp; Custom Elements for Future Web Applications.

```shell
npm install -g polymer-cli
polymer serve
```

```shell
sudo npm install -g serve
#或
yarn global add serve

# 启动静态服务
serve -s dist 

es-dev-server
```
